# Publishing Themes and Theme Sets

You can publish your themes and theme sets to make them available for use.



## Prerequisites

-   You have the `Publisher (sap-theming)` role as described in [Permissions](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/c3fdb69e274f4d9bb33139763faa7a1f.html "UI theme designer defines different permissions, that administrators can set up access for editing themes.") :arrow_upper_right:.
-   You are on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/e3585083d5ef4478b455d8773d5b25c0.html "On the welcome page, you can manage themes and theme sets - create new ones, edit, publish, export, delete, rename, duplicate and migrate them.") :arrow_upper_right:.
-   You have created a theme or theme set as described in [Create and Edit Themes and Theme Sets](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/0d2d662651d443288b5dce463acf4193.html "As a designer, you can create themes and theme sets to apply your brand to SAP Build Work Zone sites and applications.") :arrow_upper_right:.



## Context

-   Only published themes are available in the Theme Manager of SAP Build Work Zone or can be applied to applications.

    > ### Note:  
    > The Theme Manager is only relevant if you use SAP Build Work Zone. Other scenarios may require different theming tools.

-   Every time you make changes to a theme, you need to publish it again to make those changes available.



## Procedure

1.  Select the theme or theme set you want to publish.

2.  Choose *Publish*.




## Results

Your theme or theme set can now be applied to your SAP Build Work Zone sites and applications.



## Next Steps

Apply your theme or theme set as described in Applying Themes.

> ### Note:  
> You can also publish a theme in the theme menu of the editor under *Theme* \> *Save&Publish*, as described in [Customizing Themes in the Editor](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/aa4b233eb1da4f30ae6d888424a92de5.html "In the editor you can make changes to themes and preview them.") :arrow_upper_right:.
