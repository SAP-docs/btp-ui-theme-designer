# Quick Theming

With quick theming, you can easily update the color scheme, background image, or logo of your theme.



<a name="copy3f2fb6a2bb5045b58fd8b71679f07827__prereq_ycq_th3_3yb"/>

## Prerequisites

You are working in the editor as described in [Customizing Themes in the Editor](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/aa4b233eb1da4f30ae6d888424a92de5.html "In the editor you can make changes to themes and preview them.") :arrow_upper_right:.



<a name="copy3f2fb6a2bb5045b58fd8b71679f07827__context_N10015_N10012_N10001"/>

## Context

Quick theming allows you to rapidly apply your brand to your SAP Build Work Zone site and applications by adjusting main colors, the logo, and the shell background image without detailed color fine-tuning at the control level.

For more information about SAP’s theming concept, see [SAP Design System: Theming](https://www.sap.com/design-system/fiori-design-web/foundations/visual/theming). For details on the theming parameter hierarchy, refer to [SAP Design System: Design Tokens](https://www.sap.com/design-system/fiori-design-web/v1-142/foundations/visual/design-tokens) and [Theme Parameter Dependencies](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/18d80b4f0aec482a98bbd9a8bc289749.html "") :arrow_upper_right:.

> ### Note:  
> The recommended preview pages for quick theming are *Quick Theming \(SAPUI5 Applications\)* and *Launchpad \(SAPUI5 Applications\)*. To learn more about working with preview pages, see Best Practices for Selecting Preview Pages in [Selecting and Adding Preview Pages](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/898848329cd344dd8afff5dbc13c2b85.html "When you open the editor for the first time, it shows a default selection of preview pages. You can add more preview pages from a set of predefined options or add your own applications.") :arrow_upper_right:.



## Procedure

1.  In the *Editor* pane, select *Quick*.

2.  Change the parameter values according to your needs as described in [Changing Parameters](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/0d328f8ccef64def9f6cbba5e445870a.html "You can change parameters of different types, like colors or images.") :arrow_upper_right:.




## Results

The preview updates automatically.
