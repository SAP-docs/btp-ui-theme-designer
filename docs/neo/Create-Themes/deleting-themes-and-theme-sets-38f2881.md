# Deleting Themes and Theme Sets

You can delete themes and theme sets.



## Prerequisites

-   You are on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/e3585083d5ef4478b455d8773d5b25c0.html "On the welcome page, you can manage themes and theme sets - create new ones, edit, publish, export, delete, rename, duplicate and migrate them.") :arrow_upper_right:.
-   You have created a theme or theme set as described in [Create and Edit Themes and Theme Sets](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/0d2d662651d443288b5dce463acf4193.html "As a designer, you can create themes and theme sets to apply your brand to SAP Build Work Zone sites and applications.") :arrow_upper_right:.

> ### Note:  
> -   Deletion is permanent and can’t be undone.
> -   If you delete a published theme or theme set, end users who selected it experience a fallback as described in [Theme Fallback](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/bfe0aa3a3a6b4c46bc9c4b0b86b67dbb.html "Theme fallback is a multi‑phase mechanism that ensures a valid theme by checking custom, SAP, flavor, and default options when the requested theme is unavailable or invalid.") :arrow_upper_right:.



## Procedure

1.  Select the theme or theme set you want to delete.
2.  Choose *Delete*.
3.  Choose *Yes* to confirm.

> ### Note:  
> You can’t delete a theme that is used as an appearance in a theme set. To delete the theme, first edit the theme set to remove the reference to the theme as described in [Customizing Theme Sets](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/ca5d6beed238406bafa899420f03469b.html "You can create theme sets to define light, dark, and high‑contrast appearances for your theme.") :arrow_upper_right:, then publish it as described in [Publishing Themes and Theme Sets](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/f4889a4cf6714166a2fbb8b1d9eca3b2.html "You can publish your themes and theme sets to make them available for use.") :arrow_upper_right:.



## Results

The theme or theme set is deleted. It can no longer be used in SAP Build Work Zone or applications, it is no longer available in the SAP Build Work Zone theme manager, and it can't be restored.
