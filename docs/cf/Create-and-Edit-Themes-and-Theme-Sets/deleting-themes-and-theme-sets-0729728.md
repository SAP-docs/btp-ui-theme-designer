# Deleting Themes and Theme Sets

You can delete themes and theme sets.



## Prerequisites

-   You are on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](managing-themes-and-theme-sets-on-the-welcome-page-e358508.md).
-   You have created a theme or theme set as described in [Create and Edit Themes and Theme Sets](create-and-edit-themes-and-theme-sets-0d2d662.md).

> ### Note:  
> -   Deletion is permanent and can’t be undone.
> -   If you delete a published theme or theme set, end users who selected it experience a fallback as described in [Theme Fallback](../Advanced-Information/theme-fallback-bfe0aa3.md).



## Procedure

1.  Select the theme or theme set you want to delete.
2.  Choose *Delete*.
3.  Choose *Yes* to confirm.

> ### Note:  
> You can’t delete a theme that is used as an appearance in a theme set. To delete the theme, first edit the theme set to remove the reference to the theme as described in [Customizing Theme Sets](customizing-theme-sets-ca5d6be.md), then publish it as described in [Publishing Themes and Theme Sets](publishing-themes-and-theme-sets-f4889a4.md).



## Results

The theme or theme set is deleted. It can no longer be used in SAP Build Work Zone or applications, it is no longer available in the SAP Build Work Zone theme manager, and it can't be restored.
