# Create and Edit Themes and Theme Sets

As a designer, you can create themes and theme sets to apply your brand to SAP Build Work Zone sites and applications.



This section provides detailed step by step instructions for creating a new theme or theme set and editing existing ones. For more details about themes and theme sets see [What is a Theme?](../About-Themes/what-is-a-theme-71bcf85.md) and [What is a Theme Set?](../About-Themes/what-is-a-theme-set-53de0b1.md)



## Prerequisites

You have launched the theme designer as described in [Launching the Theme Designer Web App](launching-the-theme-designer-web-app-8f49dce.md).



## Procedure

**For themes:**

To create a theme, follow these steps:

1.  Create a theme using one of the following methods:
    1.  Creating a new theme based on an SAP theme as described in [Creating New Themes and Theme Sets](creating-new-themes-and-theme-sets-72c730b.md).
    2.  Duplicating an existing theme as described in [Duplicating Themes and Theme Sets](duplicating-themes-and-theme-sets-ec77044.md).
    3.  Migrating an existing theme as described in [Migrating Themes](migrating-themes-cb63341.md).
    4.  Importing a theme as described in [Importing Themes and Theme Sets](importing-themes-and-theme-sets-5e3c430.md).

2.  **Optional** Add preview pages to the available preview pages list as described in [Selecting and Adding Preview Pages](Customizing-Themes-in-the-Editor/selecting-and-adding-preview-pages-8988483.md).
3.  **Optional** Define your color palette as described in [Reusing Color Parameters](Customizing-Themes-in-the-Editor/reusing-color-parameters-16ed37f.md).
4.  Change parameters as described in [Changing Parameters](Customizing-Themes-in-the-Editor/changing-parameters-0d328f8.md).
5.  Save the theme as described in [Saving Themes](Customizing-Themes-in-the-Editor/saving-themes-c99d9df.md).
6.  Publish the theme to make it available to your SAP Build Work Zone sites and applications as described in [Publishing Themes and Theme Sets](publishing-themes-and-theme-sets-f4889a4.md).

**For theme sets:**

To create a theme set, follow these steps:

1.  Create a theme set using one of the following methods:
    1.  Creating a new theme set as described in [Creating New Themes and Theme Sets](creating-new-themes-and-theme-sets-72c730b.md).
    2.  Duplicating an existing theme set as described in [Duplicating Themes and Theme Sets](duplicating-themes-and-theme-sets-ec77044.md).
    3.  Importing a theme set as described in [Importing Themes and Theme Sets](importing-themes-and-theme-sets-5e3c430.md).

2.  Define light, dark and high-contrast appearance and save it as described in [Customizing Theme Sets](customizing-theme-sets-ca5d6be.md).
3.  Publish the theme set to make it available to SAP Build Work Zone sites and applications as described in [Publishing Themes and Theme Sets](publishing-themes-and-theme-sets-f4889a4.md).
