# Detailed Theming

Detailed theming offers more main parameters than [Quick Theming](quick-theming-e4d2a95.md). It also includes key options for control-specific theming.



<a name="loiodb09bb96b1304c50a31978a48ed11506__prereq_e2m_43g_zhb"/>

## Prerequisites

You are working in the editor as described in [Customizing Themes in the Editor](customizing-themes-in-the-editor-aa4b233.md).



## Context

If you need more customization than [Quick Theming](quick-theming-e4d2a95.md) provides, but you do not want to adjust every control individually, use *Detailed* theming. This option lets you start by adjusting main parameters. You can then refine key controls as needed. The parameters are organized to support this workflow.

For more information about SAP’s theming concept, see [SAP Design System: Theming](https://www.sap.com/design-system/fiori-design-web/foundations/visual/theming). For details on the theming parameter hierarchy, refer to [SAP Design System: Design Tokens](https://www.sap.com/design-system/fiori-design-web/v1-142/foundations/visual/design-tokens) and [Theme Parameter Dependencies](../../Advanced-Information/theme-parameter-dependencies-18d80b4.md).



## Procedure

1.  In the *Editor* pane, select *Detailed*.

2.  Change the parameter values according to your needs as described in [Changing Parameters](changing-parameters-0d328f8.md).




## Results

The preview updates automatically.



## Next Steps

Save your theme as described in [Saving Themes](saving-themes-c99d9df.md). Or explore a more detailed theming approach with [Expert Theming](expert-theming-7b01607.md).

> ### Note:  
> The recommended preview pages for detailed theming are *Quick Theming \(SAPUI5 Applications\)*, *Launchpad \(SAPUI5 Applications\)* and additional control preview pages for the specific controls you want to adjust. To learn more about working with preview pages, see Best Practices for Selecting Preview Pages in [Selecting and Adding Preview Pages](selecting-and-adding-preview-pages-8988483.md).
