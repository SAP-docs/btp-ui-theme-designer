# Quick Theming

With quick theming, you can easily update the color scheme, background image, or logo of your theme.



<a name="loioe4d2a9504f614acbae461161984b6868__prereq_ycq_th3_3yb"/>

## Prerequisites

You are working in the editor as described in [Customizing Themes in the Editor](customizing-themes-in-the-editor-aa4b233.md).



<a name="loioe4d2a9504f614acbae461161984b6868__context_N10015_N10012_N10001"/>

## Context

Quick theming allows you to rapidly apply your brand to your SAP Build Work Zone site and applications by adjusting main colors, the logo, and the shell background image without detailed color fine-tuning at the control level.

For more information about SAP’s theming concept, see [SAP Design System: Theming](https://www.sap.com/design-system/fiori-design-web/foundations/visual/theming). For details on the theming parameter hierarchy, refer to [SAP Design System: Design Tokens](https://www.sap.com/design-system/fiori-design-web/v1-142/foundations/visual/design-tokens) and [Theme Parameter Dependencies](../../Advanced-Information/theme-parameter-dependencies-18d80b4.md).

> ### Note:  
> The recommended preview pages for quick theming are *Quick Theming \(SAPUI5 Applications\)* and *Launchpad \(SAPUI5 Applications\)*. To learn more about working with preview pages, see Best Practices for Selecting Preview Pages in [Selecting and Adding Preview Pages](selecting-and-adding-preview-pages-8988483.md).



## Procedure

1.  In the *Editor* pane, select *Quick*.

2.  Change the parameter values according to your needs as described in [Changing Parameters](changing-parameters-0d328f8.md).




## Results

The preview updates automatically.



## Next Steps

Save your theme as described in [Saving Themes](saving-themes-c99d9df.md). Or explore a more detailed theming approach with [Detailed Theming](detailed-theming-db09bb9.md) or [Expert Theming](expert-theming-7b01607.md).
