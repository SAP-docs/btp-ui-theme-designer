# Publishing Themes and Theme Sets

You can publish your themes and theme sets to make them available for use.



## Prerequisites

-   You have the `Publisher (sap-theming)` role as described in [Permissions](../Advanced-Information/permissions-c3fdb69.md).
-   You are on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](managing-themes-and-theme-sets-on-the-welcome-page-e358508.md).
-   You have created a theme or theme set as described in [Create and Edit Themes and Theme Sets](create-and-edit-themes-and-theme-sets-0d2d662.md).



## Context

-   Only published themes are available in the Theme Manager of SAP Build Work Zone or can be applied to applications.

    > ### Note:  
    > The Theme Manager is only relevant if you use SAP Build Work Zone. Other scenarios may require different theming tools.

-   Every time you make changes to a theme, you need to publish it again to make those changes available.



## Procedure

1.  Select the theme or theme set you want to publish.

2.  Choose *Publish*.




## Results

Your theme or theme set can now be applied to your SAP Build Work Zone sites and applications.



## Next Steps

Apply your theme or theme set as described in Applying Themes.

> ### Note:  
> You can also publish a theme in the theme menu of the editor under *Theme* \> *Save&Publish*, as described in [Customizing Themes in the Editor](Customizing-Themes-in-the-Editor/customizing-themes-in-the-editor-aa4b233.md).
