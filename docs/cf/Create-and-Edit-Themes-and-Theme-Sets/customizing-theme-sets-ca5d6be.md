# Customizing Theme Sets

You can create theme sets to define light, dark, and high‑contrast appearances for your theme.



## Prerequisites

You have created a theme set as described in [Creating New Themes and Theme Sets](creating-new-themes-and-theme-sets-72c730b.md), or you edit a theme set as described in [Editing Themes and Theme Sets](editing-themes-and-theme-sets-04d4487.md).



## Context

You can customize theme sets on the welcome page by selecting a theme set and choosing *Edit*, or by creating a new theme set. In the theme set editor you can change the properties of your theme set, as well as define the light, dark and high-contrast appearances.



## Procedure

1.  Provide information about your theme set:

    -   *Theme Set ID*: the technical identifier of your theme set
    -   *Title*: the human-readable title of your theme set
    -   *Vendor* \(optional\): the human-readable name of the organization providing your theme set
    -   *Light Theme*: the appearance in a light low-contrast environment.
    -   *Dark Theme*: the appearance in a dark low-contrast environment.
    -   *Light High Contrast Theme*: the appearance in a light high-contrast environment.
    -   *Dark High Contrast Theme*: the appearance in a dark high-contrast environment.

2.  Select *Save* to save the theme set without publishing it. Select *Save & Publish* to save and publish the theme set at the same time. See [Publishing Themes and Theme Sets](publishing-themes-and-theme-sets-f4889a4.md) for details.




## Results

Your changes are saved and you can revisit them later.

If you selected *Save & Publish*, the theme set with your changes can immediately be applied to SAP Build Work Zone sites and applications.



## Next Steps

You can continue to edit your theme sets at any time.

If you select *Save & Publish*, administrators can directly enable the theme set for end user selection in the SAP Build Work Zone theme manager. Otherwise, you can publish your changes later. For more information, see [Publishing Themes and Theme Sets](publishing-themes-and-theme-sets-f4889a4.md).
