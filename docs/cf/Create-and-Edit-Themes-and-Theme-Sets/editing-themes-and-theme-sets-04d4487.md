# Editing Themes and Theme Sets

You can edit themes and theme sets which have been created with the UI theme designer.



## Prerequisites

-   You have the `Editor (sap-theming)` or `CustomCssEditor (sap-theming)` role as described in [Permissions](../Advanced-Information/permissions-c3fdb69.md).
-   You are on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](managing-themes-and-theme-sets-on-the-welcome-page-e358508.md).
-   You have created a theme or theme set as described in [Create and Edit Themes and Theme Sets](create-and-edit-themes-and-theme-sets-0d2d662.md).



## Procedure

1.  Select the theme or theme set you want to edit.

2.  Select *Edit* or double-click the theme or theme set.




## Next Steps

Edit your theme as described in [Customizing Themes in the Editor](Customizing-Themes-in-the-Editor/customizing-themes-in-the-editor-aa4b233.md), or edit your theme set as described in [Customizing Theme Sets](customizing-theme-sets-ca5d6be.md).

> ### Note:  
> After you change a theme or theme set, you must publish it to make the changes available for your SAP Build Work Zone sites and applications.
