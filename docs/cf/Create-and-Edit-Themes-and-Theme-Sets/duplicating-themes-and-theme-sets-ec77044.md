# Duplicating Themes and Theme Sets

You can create a copy of an existing theme or theme set by duplicating it.



## Prerequisites

-   You are on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](managing-themes-and-theme-sets-on-the-welcome-page-e358508.md).
-   You have created a theme or theme set as described in [Creating New Themes and Theme Sets](creating-new-themes-and-theme-sets-72c730b.md).



## Procedure

1.  Select the theme or theme set you want to duplicate.

2.  Choose *Duplicate*.

3.  Enter a new *Theme ID* or *Theme Set ID*, a *Title* and other properties, if needed. For more information about theme properties, see [Changing Theme Properties](Customizing-Themes-in-the-Editor/changing-theme-properties-9766329.md) and for more information about theme set properties, see [Customizing Theme Sets](customizing-theme-sets-ca5d6be.md).

4.  Confirm with *Duplicate*.




## Results

The selected theme is saved as a copy with the new name you gave it. You can now change both themes independently of each other.



## Next Steps

Edit the duplicated theme as described in [Editing Themes and Theme Sets](editing-themes-and-theme-sets-04d4487.md).

> ### Note:  
> You can also create a copy of an existing theme by opening it in the theme menu of the editor and choosing *Theme* \> *Save As*, as described in [Saving Themes](Customizing-Themes-in-the-Editor/saving-themes-c99d9df.md).
