# Creating New Themes and Theme Sets

You can create new themes and theme sets using the *Create a New Theme* and *Create New Theme Set* buttons on the welcome page.



## Prerequisites

You are already on the welcome page as described in [Managing Themes and Theme Sets on the Welcome Page](managing-themes-and-theme-sets-on-the-welcome-page-e358508.md).



## Procedure

**To create a theme**

1.  Select *Create a New Theme* to open the theme creation dialog.
2.  Select a SAP theme to start with. This will be the template for your theme. You will not be editing the SAP theme directly. For available SAP themes, see [Overview of SAP Theming Content](../About-Themes/overview-of-sap-theming-content-91ebfe2.md).

**To create a theme set**

1.  Select *Create New Theme Set* to open the theme set creation dialog.



## Next Steps

Edit your theme as described in [Customizing Themes in the Editor](Customizing-Themes-in-the-Editor/customizing-themes-in-the-editor-aa4b233.md), or edit your theme set as described in [Customizing Theme Sets](customizing-theme-sets-ca5d6be.md).
