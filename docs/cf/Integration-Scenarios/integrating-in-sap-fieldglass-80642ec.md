# Integrating in SAP Fieldglass

You can export your theme and request an upload to SAP Fieldglass to apply your brand.



<a name="loio80642ec6cc3c4b7cb840e17dfec8fd8d__prereq_hh1_4ty_y3b"/>

## Prerequisites

You have exported your theme as described in [Exporting Themes and Theme Sets](../Create-and-Edit-Themes-and-Theme-Sets/exporting-themes-and-theme-sets-26e5140.md). Make sure to choose *Optional Settings \(for Experts\)*. Then select *Source Files + CSS Resources* and deselect *UR* and *SAPUI5*.



<a name="loio80642ec6cc3c4b7cb840e17dfec8fd8d__steps_ih1_4ty_y3b"/>

## Procedure

Contact your SAP Fieldglass representative to upload the theme and make it available for users as described in [SAP Fieldglass: Administration Fundamentals \> User](https://help.sap.com/docs/SAP_Fieldglass/26592f9ffa0b4a9c9646db5877c5eb5f/a984d63456e64e74a9dbc3b4b2b7696e.html).
