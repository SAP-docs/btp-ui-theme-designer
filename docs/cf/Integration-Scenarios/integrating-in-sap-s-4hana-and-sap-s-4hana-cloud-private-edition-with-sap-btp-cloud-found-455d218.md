# Integrating in SAP S/4HANA and SAP S/4HANA Cloud Private Edition with SAP BTP, Cloud Foundry Themes

You can use themes created in SAP BTP, Cloud Foundry edition directly in your SAP S/4HANA or SAP S/4HANA Cloud Private Edition by integrating SAP BTP, Cloud Foundry themes with SAP S/4HANA.



## Prerequisites

You have published a theme or theme set as described in [Publishing Themes and Theme Sets](../Create-and-Edit-Themes-and-Theme-Sets/publishing-themes-and-theme-sets-f4889a4.md).



## Procedure

1.  Setup the integration to SAP BTP, Cloud Foundry themes in the SAP S/4HANA system as described in [Integrating SAP BTP, Cloud Foundry Themes with SAP S/4HANA](https://help.sap.com/docs/ABAP_PLATFORM_NEW/7b10839d7f5f45b4b2a1a96fc95fd9bb/3ccb78c6f0704016b420cf0d0fc82fa4.html?version=202510.001&state=DRAFT).

2.  Apply your theme to SAP Fiori launchpad as described in [Setting Themes for the Launchpad](https://help.sap.com/docs/ABAP_PLATFORM_NEW/a7b390faab1140c087b8926571e942b7/86c18ca550834b8eb0e27bfbdcb492db.html?version=202510.000) or apply your theme to standalone apps as described in [Applying Custom Themes to Standalone Apps](https://help.sap.com/docs/ABAP_PLATFORM_NEW/7b10839d7f5f45b4b2a1a96fc95fd9bb/e6ede69d016e41a98a6e66d3be86665e.html?state=DRAFT).
