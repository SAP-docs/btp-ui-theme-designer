# Integrating in SAP GUI for Windows

You can export your theme and deploy it in SAP S/4HANA to apply your brand to SAP GUI for Windows.



<a name="loiod026b5999e1148509d1ce670a7e6eef1__prereq_hh1_4ty_y3b"/>

## Prerequisites

You have exported your theme as described in [Exporting Themes and Theme Sets](../Create-and-Edit-Themes-and-Theme-Sets/exporting-themes-and-theme-sets-26e5140.md). Make sure to choose *Optional Settings \(for Experts\)*. Then select *Source Files + CSS Resources* and deselect *UR* and *SAPUI5*.



<a name="loiod026b5999e1148509d1ce670a7e6eef1__steps_ih1_4ty_y3b"/>

## Procedure

Deploy your theme to SAP S/4HANA as described in [SAP GUI for Windows: Custom Themes created via UI Theme Designer](https://help.sap.com/docs/sap_gui_for_windows/dfad9ecd79db404eba46fdd709013a78/3f5184e38e7f4442b737eb81c315ecf2.html).
