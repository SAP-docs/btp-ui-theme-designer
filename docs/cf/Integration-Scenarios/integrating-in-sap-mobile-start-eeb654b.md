# Integrating in SAP Mobile Start

You can export your themes and upload them in SAP Mobile Start to apply your branding.



## Prerequisites

You have exported your theme as described in [Exporting Themes and Theme Sets](../Create-and-Edit-Themes-and-Theme-Sets/exporting-themes-and-theme-sets-26e5140.md). Make sure you select *Optional Settings \(for Experts\)*. Then select *Source Files + CSS Resources* and deselect *UR* and *SAPUI5*.



## Procedure

Upload your theme in SAP Mobile Start as described in [Themes for Custom Corporate Branding](https://help.sap.com/docs/mobile-start/mobile-start-administration-guide/themes-for-custom-corporate-branding-overview?version=2.5).

 > ### Note:  
> SAP Mobile Start is a native mobile application, not a web application. Themes created with the Theme Designer web app are primarily for web applications. For detailed information about using themes with SAP Mobile Start, including restrictions related to mobile theming, refer to [Mobile Theming](../Create-and-Edit-Themes-and-Theme-Sets/Customizing-Themes-in-the-Editor/mobile-theming-6237538.md) and [Themes for Custom Corporate Branding](https://help.sap.com/docs/mobile-start/mobile-start-administration-guide/themes-for-custom-corporate-branding-overview?version=2.5).
