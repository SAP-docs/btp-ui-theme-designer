# Integration Scenarios

You can use themes created in the UI theme designer in different scenarios.

As an administrator, you can integrate themes and theme sets in SAP applications for all users or allow users to select their preferred themes. Some SAP applications enable users to choose themes via the Appearance dialog. As a developer, you can apply your custom theme to your own applications.
