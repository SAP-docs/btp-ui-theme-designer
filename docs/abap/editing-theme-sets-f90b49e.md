# Editing Theme Sets



## Procedure

1.  On the welcome screen, select the theme set you want to edit.

2.  Select *Edit* and make your changes.

3.  Choose *Save* or *Save & Publish*.

    > ### Note:  
    > To apply the theme set to an application, you must first publish it.
