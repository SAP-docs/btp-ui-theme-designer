# Changing Parameters

You can change parameters of different types, like colors or images.

While you can always enter the LESS expression of your value directly, there are dedicated pickers for the different parameter types. The preview updates automatically, and you can discard unsaved changes or reset to the original value.



## Prerequisites

You are working in the editor as described in [Customizing Themes in the Editor](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/aa4b233eb1da4f30ae6d888424a92de5.html "In the editor you can make changes to themes and preview them.") :arrow_upper_right:.



## Procedure

Use [Quick Theming](quick-theming-e4d2a95.md), [Detailed Theming](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/db09bb96b1304c50a31978a48ed11506.html "Detailed theming offers more main parameters than . It also includes key options for control-specific theming.") :arrow_upper_right:, [Mobile Theming](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/6237538bfaaa41a8b60f8c24b81083c8.html "Mobile theming displays the theming parameters supported by SAP Mobile Start.") :arrow_upper_right: or [Expert Theming](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/7b01607cb9564f6d94087a6db995c350.html "If you need to make changes beyond what or offer, use expert theming. This option lets you browse all available parameters and make fine-tuned adjustments to your theme.") :arrow_upper_right: to find the parameter you want to change.



### To change a parameter:

You can enter the desired value for parameters of all types. The value can be a LESS expression. Refer to the following pages for details for the different types:

-   [Changing Colors](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/b6900f3e9cf649d095e3b057924c4c0f.html "You can change the colors of your theme, either by using the color picker or by entering the value directly.") :arrow_upper_right: provides details on color parameters.
-   [Adding Images](adding-images-047e630.md) describes how you can upload and use images.
-   [Changing Dimensions](changing-dimensions-49c377f.md) goes into detail on dimension parameters.
-   [Changing Font Families](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/8af3480eac4944be9262309871af03ac.html "You can change font families. Use this option with caution. Changing the font family can affect text width and impact the layout of SAP Build Work Zone sites and applications.") :arrow_upper_right: shows how you can use your desired font family.
-   [Reusing Color Parameters](reusing-color-parameters-16ed37f.md) lets you define palette parameters that you can use throughout your theme.



### To discard unsaved changes:

If you have unsaved changes to a parameter, you can discard them by selecting “Discard unsaved changes” <span class="SAP-icons-V5"></span> next to the value.



### To reset to the original value:

If you have saved changes to a parameter, you can reset to the original value by selecting “Reset to original value” <span class="SAP-icons-V5"></span> next to the value.



## Result

The preview updates automatically.



## Next Steps

Save your theme as described in [Saving Themes](https://help.sap.com/viewer/09f6818d8e064537973102d6289e2aca/Cloud/en-US/c99d9df29d3c4a45a883e15ce2753178.html "You can save changes you made in your theme at any time during its creation and design.") :arrow_upper_right:.
